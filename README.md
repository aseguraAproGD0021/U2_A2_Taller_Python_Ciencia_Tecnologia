# Taller de Python para la Ciencia y la Tecnología en Ingeniería
Repositorio del taller con entrada/salida de datos, funciones, módulos, numpy, pandas y matplotlib.

## Cuaderno de trabajo (Google Colab / Jupyter)

El cuaderno requerido por el taller se llama **TallerPython_SeguraAdrian.ipynb** y se encuentra dentro de la carpeta `notebooks/`.  
Este archivo incluye todos los ejercicios solicitados en el taller:

- Variables y tipos de datos  
- Conversión de datos (`int()`, `float()`)  
- Entrada y salida de datos con `input()`  
- Estructuras de control (if, for, while)  
- Funciones definidas por el usuario  
- Uso básico de módulos y paquetes  
- Numpy: arreglos, operaciones y ejemplos aplicados  
- Pandas: lectura y análisis de datos  
- Matplotlib: gráficas básicas  
- Ejemplos prácticos orientados a la ciencia y la ingeniería  
- Comentarios y encabezados explicativos en cada sección
